What is ReAct?

// Reason + Act — a prompting strategy that interleaves thinking and tool use in a loop.
🧠
Reason
LLM introspects: what do I know? what do I need? It writes out its reasoning before taking any action.
⚡
Act
Calls a tool, API, or search with specific parameters — database query, web lookup, external service.
👁
Observe
Reads the result and adds it to context. The model now knows more than it did before the action.
🔁
Repeat
Loops until the goal is fully achieved or max iterations reached — enabling multi-step agentic tasks.
