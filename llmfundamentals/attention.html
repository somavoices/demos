Why Attention Exists
// a brief history of the mechanism powering every modern LLM
1
The RNN Problem
RNNs processed text one token at a time, passing meaning through a single fixed-size vector. By token 50, the signal from token 1 was largely forgotten. Long-range dependencies were nearly impossible.
Bottleneck: all context squeezed through one vector — early words vanished.
→
2
2017 Breakthrough
Vaswani et al. (Google, 2017) — throw away recurrence entirely. Let every token look directly at every other token simultaneously, weighted by relevance. No bottleneck. Token 1 and token 50 communicate directly.
Key insight: direct all-to-all lookup. Distance no longer equals information loss.
→
3
QKV — the formula
softmax( QKᵀ / √dₖ ) V
Q — "what am I looking for?" K — "what do I offer?" V — "what I give you"
Intuition: "its" queries → "loan" matches → "loan"'s value dominates the output.
→
4
Why "Attention"?
Borrowed from cognitive psychology — selective attention: humans don't process all input equally; we focus on what's relevant and suppress the rest. The mechanism is the model's differentiable version of that selective focus.
Psychology → ML: attention weights = learned, end-to-end optimized selective focus.
